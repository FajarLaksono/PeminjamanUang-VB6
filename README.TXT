===========================
- Program Peminjaman Uang -
===========================

Program peminjaman uang, program ini ditujukan untuk kasir dari organisasi peminjaman uang.
calon peminjam harus mendaftar terlebih dahulu sebagai anggota dari organisasi agar dapat meminjam uang.

pengembalian uang dapat dilakukan beberapa kali.

Software : VB6 SP6, xampp atau lainya.

=============
- Warning ! -
============= 
Please do not modify structure of directory.

dan 

Program ini membutuhkan 
=> File :
	MSCOMCTL.ocx
	MSCOMCT2.ocx

Kemungkinan beberapa fitur control dari MSCOMCTL.ocx dan MSCOMCTL.ocx akan hilang ketika pertama kali anda buka 
project ini dalam VB anda. 

Silahkan buat ulang control yang hilang tersebut. biaasanya control tersebut akan berubah menjadi picturebox
beberapa control yang kemungkinan hilang adalah

- strbar(Status Bar) pada frmMenuUtama.frm
- iconList(ImageList) pada frmMenuUtama.frm
- Toolbar (Toolbar) pada frmMenuUtama.frm
